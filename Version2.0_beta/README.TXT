程序名称

         图像信息隐藏系统

版本号
 
         V2.0 beta

功能描述

           图像信息隐藏系统是使用matlab语言编写的一款可以将秘密信息隐藏
到图像的工具。

           本系统可以使用四种算法进行信息隐藏，分别是空间域的LSB、LHA
和变换域的DCT、DWT。本系统还设计了对称密钥系统，以保证系统的安全
特性。

           本系统设计了噪声攻击、滤波处理和其他一些攻击，可以通过这些攻击
手段对四种算法的鲁棒性加以验证。

           本系统可以对嵌入秘密信息后的载体图像和提前出来的秘密信息进行统
计处理，得到相关性能的数据。

           本系统目前只能使用灰度图像作为载体图像，需要使用RGB图像作为
信息载体图像，可以稍作修改即可支持。

           本系统可以嵌入二值图像和文字。

           为了保证能够正常使用本系统，请使用Matlab 2009a打开。具体步骤
为：打开Matlab――File――New――GUI――Open Existing GUI
――Browse，找到info_hiding_gui.fig文件打开即可。

           最后声明一点，本系统凝聚了本人一个多月来的心血，可以供同学们内
部学习交流使用，希望大家不要用于其它用途。
